# 华夏文脉 · 文化旅游网站

以 **地域文化** 和 **历史文化** 为主题的旅游介绍网站，带你领略中华文明的多元魅力。

## 站点概览

| 页面 | 简介 |
|------|------|
| **首页** | 全屏 Hero、6 个目的地卡片、8 项文化印记、历史年表 |
| **北京 · 帝都风华** | 故宫、长城、天坛、胡同文化、北京美食 |
| **西安 · 千年古都** | 兵马俑、大雁塔、古城墙、回民街、西北美食 |
| **敦煌 · 丝路明珠** | 莫高窟、鸣沙山、玉门关、雅丹地貌、丝路美食 |

## 技术栈

- 原生 HTML5 + CSS3 + JavaScript
- 纯静态站点，无需构建工具
- 响应式设计，适配桌面与移动端

## 访问链接

**https://justit0202.github.io/huaxia-wenmai/**

## 本地预览

```bash
python -m http.server 8080
# 或
npx serve .
```

## 特色

- 暖色调朱红 + 金色配色，契合中式美学
- 沉浸式全幅 Hero 视觉
- Intersection Observer 驱动滚动动画
- 移动端自适应布局
