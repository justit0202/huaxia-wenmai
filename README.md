# 前詹·印象 — 滨海小镇·渔乡风情

惠来县 **前詹镇** 的介绍型旅游网站，以地域文化、历史文化和美食文化为主题。

## 站点概览

| 页面 | 简介 |
|------|------|
| **首页** | 全屏 Hero、6 大特色卡片、8 项文化印记、历史年表 |
| **历史沿革** | 前詹的起源与发展，从古代到新时代 |
| **自然风光** | 海岸、渔港、田园，南海之滨的壮美 |
| **美食文化** | 潮汕海鲜、地方小吃、工夫茶文化 |

## 技术栈

- 原生 HTML5 + CSS3 + JavaScript
- 纯静态站点，无需构建工具
- 响应式设计，适配桌面与移动端

## 访问链接

**https://justit0202.github.io/huaxia-wenmai/**

## 本地预览

```bash
python -m http.server 8080
# 或
npx serve .
```

## 特色

- 滨海风情主题，朱红 + 金色中式美学
- 沉浸式全幅 Hero 视觉
- Intersection Observer 驱动滚动动画
- 移动端自适应布局
